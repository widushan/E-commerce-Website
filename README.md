# 🛒 Trendzide E-commerce Website

A modern, full-stack e-commerce platform for fashion and lifestyle products. Built with React, Node.js, Express, and MongoDB.

---

## 🚀 Features
- User authentication & admin panel
- Product catalog with images, categories, and sizes
- Shopping cart & checkout
- Order management
- Responsive design
- Image upload (Cloudinary)
- Payment integration (Stripe, Razorpay)

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Cloud:** Cloudinary
- **Payments:** Stripe, Razorpay

---

## 📦 Installation & Setup

```bash
# Clone the repository
https://github.com/widushan/E-commerce-Website.git

# Install dependencies for backend
cd backend
npm install

# Install dependencies for frontend
cd ../frontend
npm install

# Install dependencies for admin panel
cd ../admin
npm install
```

---

## ▶️ Usage

1. **Start Backend**
	```bash
	cd backend
	npm start
	```
2. **Start Frontend**
	```bash
	cd frontend
	npm run dev
	```
3. **Start Admin Panel**
	```bash
	cd admin
	npm run dev
	```
